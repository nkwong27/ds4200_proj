Cumulative Returns Before and After Major Economic Events

Visualization Details

  • X-axis: Days from event start (0 = event start date)
  • Y-axis: Cumulative Return (%) (represents the total gain or loss of an investment over a specific period)
  • Lines: Average cumulative returns by company rank group
  • Vertical line: Event start date (Day 0)

Rank Groups: Companies are grouped by market cap ranking (Top 1-10, 11-20, 21-30, 31-40, 41-50, 51+)
